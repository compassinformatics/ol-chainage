# ol-chainage

Adds chainage markers to an OpenLayers geometry.

![Line geometry with chainage markers every 10m](example.png)

## Build

To generate a build ready for production:

```
npm install
npm run build
```

## Example

To run:

```
cd example
npm install
npm start
# http://localhost: 1871
```


To build:

```
cd example
npm run build
```

Then deploy the contents of the `dist` directory to your server. 
You can also run `npm run serve` to serve the results of the `dist` directory for preview.
